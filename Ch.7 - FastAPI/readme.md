1. What is FastAPI ?

> A modern, high-performance web framework for building APIs with Python based on standard type hints

2. What Feature is in FastAPI ?

> Fast to run  
> Fast to code  
> Reduced number of bugs  
> Intuitive  
> Straightforward  
> Short  
> Robust  
> Standards-based  
> Have a lot of automatic docs
